# studying IPC in Linux
Репозиторий посвящен изучению основных методов межпроцессорного взаимодействия в ОС Linux.
1) Неименновае и именованные каналы
2) Очереди сообщений
3) Разделяемая память
~~4) Сигналы~~
~~5) Сокеты~~
~~6) D-Bus~~

Реализована одинаковая задача межпроцессорного взаимодействия, но разными методами. Процесс-клиент отправляет процессу-серверу строку, которую введет пользователь, в свою очередь процесс-сервер получит ее, обработает(в данной реализации просто перевернет) и отправит обратно процессу-клиенту.
